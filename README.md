# MEMORY ROOM

## About the Project

MEMORY ROOM is a Unity VR prototype focused on memory and object interaction inside an immersive environment.

At the beginning of the experience, the player has 7 seconds to memorize all the objects placed in the room. After the memorization phase ends, the player can interact with the environment and pick up any object simply by moving over it and grabbing it through the current VR interaction system.

For testing purposes inside the Unity editor, objects can temporarily be dropped using the **E** key on the keyboard. In the final VR version, this mechanic will be replaced by a more natural interaction system based on the height and movement of the VR sensors/controllers.

---

## Current State

The project is currently in an early prototype phase where the main focus is building the core gameplay loop and interaction systems. At the moment, the game already includes a functional object pickup system, temporary object dropping, chest opening animations, basic VR interactions, and the first version of the gameplay flow. The goal of this phase is to validate the interaction mechanics and create a solid technical foundation before expanding the experience.

---

## Final Goal

The final objective of MEMORY ROOM is to become a complete VR memory and puzzle experience focused on immersion and natural interaction. The project aims to include progressive difficulty, more complex memory challenges, physics-based interactions, environmental storytelling, and fully gesture-based mechanics without relying on keyboard inputs. The final experience should feel intuitive, immersive, and fully adapted to virtual reality gameplay.
